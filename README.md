# Real Estate Marketplace

## Overview
The Real Estate Marketplace is a web application designed to connect buyers, sellers, and real estate professionals in Senegal. The platform provides comprehensive services for property transactions, including buying, selling, renting, and securing home loans.

## Features
- **Deukouway Immo:** Buy, sell, and rent properties.
- **Home Loans:** Financial services for property purchases.
- **Find Agents:** Connect with real estate agencies.
- **Property Trends:** Stay updated with market trends.
- **Key Participants:** Information about notaries, photographers, home builders, and more.
